# Where do I create tags?

Tags are created while editing a recipe, not in Settings.

**To create a new tag:**
1. Open any recipe and tap **Edit**
2. Scroll to the **Tags** section
3. Start typing a new tag name
4. Select it to create and apply the tag

**To manage existing tags:**
Go to **Settings > Manage Tags** where you can change tag colors or delete tags you no longer need.
