# StockpotのAI機能

Stockpotは、レシピの取り込みと作成をより速くするためにAIを活用しています。できることは以下の通りです：

## ソーシャルからインポート

Instagram、TikTok、YouTubeのレシピ動画をStockpotに直接共有できます。投稿の説明からレシピを抽出します。

## URLからインポート

レシピのリンクを貼り付けると、完全なレシピを抽出します。ほとんどのレシピサイトやブログで動作します。

## AIで生成

食べたいものを説明すると、レシピのアイデアを取得できます。パントリーの食材を含めることで、手元にある材料に基づいた提案も可能です。

## クリッピングから抽出

レシピのテキストをクリッピングとして保存した場合、ワンタップで構造化されたレシピに変換できます。買い物リストの抽出にも対応しています。

## Discover

アプリ内から人気のレシピサイトを閲覧し、見つけたレシピをインポートできます。

AI機能にはPlusサブスクリプションが必要です。
