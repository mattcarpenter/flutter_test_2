# クッキングモードの機能

クッキングモードは、キッチンでレシピを見ながら調理するための集中できる表示方法を提供します。

## 調理を開始する

レシピを開いて **調理を開始** をタップします。一度に1つの手順が大きく読みやすいテキストで表示されます。**前へ** と **次へ** ボタンで手順を移動できます。

注：セクションヘッダー（「ソースの材料」など）は表示されますが、手順としてはカウントされません。

## 複数のレシピを同時に調理

複数の料理を一度に作る場合、クッキングセッションにレシピを追加できます：

1. クッキングモード中に **レシピを追加** をタップ
2. 別のレシピを選択
3. 画面下部にレシピカードが表示されます
4. カードをタップしてレシピを切り替え

各レシピは個別に進捗を追跡するため、どこまで進んだか見失うことはありません。

## 一時停止と再開

進捗は自動的に保存されます。離席する必要がある場合：
- クッキングモードまたはアプリを閉じるだけ
- 後でレシピに戻る
- **調理を再開** をタップして続きから再開
