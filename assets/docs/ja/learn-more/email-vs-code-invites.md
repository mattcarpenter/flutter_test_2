# メール招待とコード招待の違い

世帯に誰かを招待する方法は2つあります。

## メール招待

- 相手のメールアドレスを入力
- 参加方法の説明が記載されたメールが届きます
- 正確なメールアドレスがわかっている場合に最適

## コード/リンク招待

- 共有可能なリンクまたはコードを生成
- リンクを持っている人は誰でも参加可能
- テキストメッセージやSNSなどで共有する場合に最適
- 「田中さんへの招待」のようなラベルを追加して、誰に送ったか追跡可能

## 知っておくと便利なこと

- どちらの方法も、承諾後は同じように機能します
- 招待リンクはセキュリティ上の理由で一定期間後に期限切れになります
- 招待される人は、すでに別の世帯に属していてはいけません（一度に1つの世帯にのみ所属可能）
