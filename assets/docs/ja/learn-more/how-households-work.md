# 世帯の仕組み

世帯とは、レシピデータを共有するグループです。一緒に料理をする家族やルームメイトに最適です。

## 共有されるもの

世帯に所属すると、全員が以下を共有します：
- すべてのレシピとフォルダ
- パントリー全体
- すべての買い物リスト
- すべての献立

## 個人のままのもの

以下は各個人に固有のままです：
- アカウント設定
- アプリの設定（テーマやフォントサイズなど）

## 重要なルール

- **一度に1つの世帯** にのみ所属できます
- 世帯を作成した人が **オーナー** になります
- オーナーはメンバーの招待と削除ができます
- すべてのメンバーは共有データを表示・編集できます

## 世帯を離れる

- 一般メンバーの場合、いつでも離れることができます
- オーナーの場合、先に別のメンバーにオーナー権限を移譲する必要があります
- 唯一のメンバーの場合、離れると世帯が削除されます

離れると、レシピは個人用（非共有）になります。削除されることはありません。
