# 在庫ステータスの説明

パントリーアイテムには、レシピマッチングに影響し、買う必要があるものを追跡するのに役立つ3つの在庫レベルがあります。

## 3つのレベル

**在庫あり**（緑）
- 持っていて、使用可能な状態
- レシピの材料マッチングにカウントされます

**残りわずか**（オレンジ）
- 残り少ない - 早めに買い足すリマインダー
- レシピの材料マッチングには引き続きカウントされます

**在庫切れ**（赤）
- 完全になくなった状態
- レシピの材料マッチングにはカウントされません

## 各ステータスの使い分け

- 残り少なくなっていることに気づいたら **残りわずか** にマーク
- 完全になくなったら **在庫切れ** にマーク
- 買い物後に **在庫あり** に更新

**ヒント：** 買い物後に買い物リストから「パントリーを更新」機能を使うと、購入したアイテムをすばやく「在庫あり」にマークできます。
