# 購入したアイテムをパントリーに移動するには？

買い物をしてリストのアイテムにチェックを入れた後：

1. アイテムをタップして購入済みとしてチェック
2. 右下に **「チェック済みを...」** ボタンが表示されます
3. タップして **パントリーを更新** を選択
4. アイテムを確認します - 2つのセクションが表示されます：
   - **追加するアイテム：** パントリーに新規作成されるアイテム
   - **更新するアイテム：** 「在庫あり」にマークされる既存のパントリーアイテム
5. 更新したくないものはチェックを外す
6. **パントリーを更新** をタップして確認

パントリーが更新され、チェック済みのアイテムは買い物リストから削除されます。
