# パントリーに追加したのにレシピの材料とマッチしない

これは通常、材料とパントリーアイテムが一致する **エイリアス** を共有していない場合に発生します。

**エイリアスとは？**

エイリアスは材料をマッチングするための別名です。例えば、「マヨネーズ」というパントリーアイテムに「マヨ」と「マヨネーズ」というエイリアスがあれば、どちらの名前を使うレシピの材料ともマッチできます。

**トラブルシューティング手順：**

1. **インターネット接続を確認**
   アイテムを追加するとエイリアスが自動生成されますが、これにはインターネット接続が必要です。オフライン中にアイテムを追加した場合は、オンラインに戻るまで待ってください。

2. **パントリーアイテムのエイリアスを確認**
   - パントリーに移動
   - アイテムをタップして編集
   - **エイリアス** セクションを確認
   - 不足している場合は材料名をエイリアスとして追加

3. **レシピの材料のエイリアスを確認**
   - レシピを開いて材料リストを表示
   - 材料の横にある色付きステータスチップをタップ（ある場合）
   - 材料をタップしてエイリアスを確認
   - 必要に応じてパントリーアイテム名をエイリアスとして追加

**ヒント：**
- パントリーアイテムには一般的で汎用的な名前を使用（「2%オーガニック牛乳」ではなく「牛乳」）
- エイリアスは大文字小文字を区別しません（「牛乳」と「Milk」はマッチ）
- エイリアスは好きなだけ追加できます
