# レシピのインポートがうまくいかない

レシピのインポートに問題がある場合、確認すべきことがいくつかあります。

**レシピページにいることを確認**

インポートは実際にレシピが含まれているページでのみ機能します。ホームページ、カテゴリページ、検索結果では機能しません。個別のレシピページにいる必要があります。

**抽出の仕組み**

まず、ページから構造化されたレシピデータの抽出を試みます。それが利用できない場合は、AIを使用してレシピを見つけて抽出します。一部のページには、どちらの方法でも機能するのに十分な情報がない場合があります。

**代わりにクリッピングを試す**

インポートが失敗し続ける場合は、レシピのテキストをコピーして新しいクリッピングに貼り付けてみてください。その後、**レシピに変換** を使用して抽出できます。

**接続を確認**

レシピのインポートにはインターネット接続が必要です。何もロードされない場合は、オンラインであることを確認してください。
