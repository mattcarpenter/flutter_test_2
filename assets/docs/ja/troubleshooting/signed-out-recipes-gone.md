# サインアウトしたらレシピが全部消えました

ご心配なく - レシピはクラウドに安全に保存されています！

**解決方法：** 同じアカウントで再度サインインしてください。すべてのレシピ、フォルダ、パントリーアイテム、献立が再び表示されます。

**なぜこうなるのか：**

サインアウトすると、アプリはデータのない「ゲスト」モードに切り替わります。再度サインインすると、個人データがクラウドから同期されます。

**同期に関する重要な注意：**

データの同期が完了する前にサインアウトすると、最近の変更が失われる可能性があります。「一部のデータの同期が完了していません」のようなメッセージが表示されたら、サインアウトする前に完了するまで待ってください。
